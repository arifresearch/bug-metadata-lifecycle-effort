# Replication Package

This repository contains the complete replication package for the study:

**Bug Metadata as Low-Fidelity Proxies for Maintenance Effort**

## Study Overview

This replication package reproduces all statistical analyses, tables, and figures reported in the manuscript, including:

- Descriptive statistics
- Skewness analysis
- Spearman correlation analysis
- Kruskal-Wallis group comparisons
- Multiple linear regression (log-transformed BLE)
- All generated figures

All analyses are deterministic.
